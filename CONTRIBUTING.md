# Contributing

Thank you for helping improve this translation! Contributions are always welcome.

## Reporting issues or suggestions

Found a typo, a missing translation, or have an idea for improvement? Please:
- Open an **issue** or **discussion** on this repository.
- Submit a **pull request** for translation fixes or improvements.
